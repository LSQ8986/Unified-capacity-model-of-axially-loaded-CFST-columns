Unified capacity model of axially loaded CFST with different sectional shapes

Parameter Reference

fy: Steel strength (MPa) fc: Concrete strength (MPa) b: Sectional width (mm) h: Sectional height (mm) t: Steel thickness (mm) r0: Outer corner radii (mm) L: Column length (mm)

Single Calculation

Parameters input [fy, fc, b, h, t, r0, L]

Batch Calculation

Parameters input [fy, fc, b, h, t, r0, L]

Supported formats: CSV, Excel, TXT
fy /MPafc /MPab /mmh /mm> t /mmr0 /mmL /mmNu /kN

The model was developed based on the following references:

[1] Siqi Lin, Yan-Gang Zhao, Zhao-Hui Lu, Xi-Feng Yan. Unified Theoretical Model for Axially Loaded Concrete-Filled Steel Tube Stub Columns with Different Cross-Sectional Shapes. Journal of Structural Engineering-ASCE, 2021, 147(10): 04021159.

[2] Siqi Lin, Yan-Gang Zhao, Zhao-hui Lu. Fibre beam element models for nonlinear analysis of concentrically loaded circular CFT columns considering the size effect. Engineering Structures, 2020, 210: 110400.

Note: The model in Ref. [1] was extended by incorporating the slenderness model from Ref. [2], so the model on this webpage also applies to medium and long columns.
